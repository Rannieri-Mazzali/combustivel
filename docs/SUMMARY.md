## 🚀 Como Começar

### Passo 1: Abrir a Documentação (2 minutos)
Abra o arquivo `INDEX.html` em seu navegador para ter uma visão visual do projeto.

### Passo 2: Configurar Firebase (10 minutos)
1. Vá para https://console.firebase.google.com
2. Crie projeto chamado "FleetFuel"
3. Copie suas credenciais
4. Cole em `src/js/firebase-config.js`

### Passo 3: Configurar Email (5 minutos)
1. Registre em https://www.emailjs.com
2. Configure seu email service
3. Copie credenciais
4. Cole em `src/js/modules/email.js`

### Passo 4: Testar Localmente
```bash
cd public
python -m http.server 8000
# Acesse http://localhost:8000
```

### Passo 5: Deploy (Opcional, 10 minutos)
```bash
npm install -g firebase-tools
firebase login
firebase deploy
```
